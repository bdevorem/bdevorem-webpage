Sentiment Wars

Sentiment Wars is a sentiment analyzer of tweets regarding 'Star Wars: The Force Awakens', with both command-line and web interfaces. In the web app, you can see a series of tweets queried using the Twitter Search API and the Tweepy wrapper, along with their sentiment analysis using a 4-state deterministic finite automaton. The actual automaton is generalized, i.e. it can accept any input in the form of multiple English words delimited by whitespace. The Sentiment Wars automaton can be downloaded from the app's Github, along with the README containing directions on how to run the automaton from the command line with user-inputted tweets, or with specific Twitter Search queries.

Lamby

Lamby is a simple Lambda Calculus Reducer, written in Python. I have developed a recent interest in Programming Language Theory (and theory of computation, just in general)(as noted in multiple places on this webpage), so this project has served as an interest-gauge for possible research areas of my future. Currently, Lamby does not support arithmetic, but the goal is to turn Lamby from a reducer to a complete interpreter. Lamby has been one of my favorite projects thus far in my education. It really gave me a sense of what sort of things I could do in graduate school, which was so.exciting.

NDLUG Webpage

A second attempt at creating a webpage... second to the one you are currently at. This time, instead of using the Flask microframework, we used Django (a huge step up in the world of webdev). The NDLUG webpage is just that, a webpage for the Notre Dame Linux User Group, a club that I have co-founded/co-benevolently-dictated. In order to lay a solid foundation for the club, we are trying to maintain as much in-house technology as possible, such as user accounts, polling and announcements apps, and archives of past meetings. Additionally, the site also acts as a stepping stone for LUG members in terms of open-source development and contribution, by being a safe project to learn about version control, documentation, markdown, maintenance, bugs/issues, pull requests, etc. Much fun.

Super-ish Mario Bros

The token C++ game! Super-ish Mario Bros is a remake of the classic Super Mario Bros game, programmed completely in C++ with the SDL library for graphics. We utilized the traditional ideas of mushrooms, goombas, and koopas, but we took a different perspective on other ideas in order to maintain uniqueness. This project was one of the more challenging, simply because I am not very passionate about graphics and the majority of time spent on this project went towards the graphical interface.
